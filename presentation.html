PBR DISTORTION
R&D 투자가 자산으로 인정받지 못하면,
장부상 자기자본(Book Equity)이 증발합니다.
이는 곧, 기업이 실제 내재가치보다 훨씬 비싸게 거래되는 것(고평가)처럼 보이는 착시를 유발합니다.
"
어떤 기업은 실제로는 미래 성장동력을 강력하게 쌓고 있는데,

착각하게 만듭니다.

"무형자산 투자를 많이 하는 기업이

ROBUSTNESS CHECK

시점 정합성 (Look-ahead Bias 방지)

미래 정보를 과거에 미리 알고 투자하는 백테스트 오류 방지
ValuationDate = max(결산 후 3개월 월말, 실제 공시일 월말)
재무정보가 시장에 공식적으로 공개되고, 투자자가 실제로 확인 가능한 시점 이후의 가격만 사용
CORE LOGIC

회계 보정 로직 (무형자산 자산화)

Investment = R&D + 0.3 × SG&A
매년 33%의 상각률을 적용하여 영구재고법으로 기업의 진짜 무형자본을 누적 계산

INPUT FEATURES

3가지 핵심 회계 팩터

1. 수익성 팩터 (m₁)
보정 자기자본 대비 순이익
m₁ = 당기순이익 / 보정 자기자본
2. 이익 왜곡 팩터 (m₂)
기존 대비 보정 영업이익 차이
m₂ = (보정 영업이익 - 기존 영업이익) / 기존 영업이익
3. 투자 가속도 팩터 (m₃)
무형투자 증가 속도
m₃ = 당기 무형투자 / 과거 2년 평균 무형투자
TARGET VARIABLE

타겟과 Gap의 의미

타겟(y) : 섹터 중립 상대 PBR
(개별 종목 PBR - 섹터 중앙값 PBR)

GAP = 예측된 상대 PBR - 실제 관측된 상대 PBR
= 예측된 상대 PBR - (실제 관측 PBR - 섹터 중앙값 PBR)
UNDERVALUATION SIGNAL

상대적 저평가 후보 (Gap)

Gap이 크다는 것은
모델 기준으로는 훨씬 높은 가치(PBR)를 인정받아야 하는데,
시장에서 부당하게 낮게 평가되고 있다는 뜻입니다.
= 회계 보정 기준의 강력한 '매수(Long) 후보'

  • Robustness: RankNet 모델의 다중 시드(Seed) 및 하이퍼파라미터 추가 튜닝 필요
